> 문제
> 두 배열이 얼마나 유사한지 확인해보려고 합니다. 문자열 배열 s1과 s2가 주어질 때 같은 원소의 개수를 return하도록 solution 함수를 완성해주세요.


```python
#내 코드
def solution(s1, s2):
    cnt = 0
    for i in s1:
        if i in s2 :
            cnt += 1
    return cnt
```

```python
# 아무개 코드
def solution(s1, s2):
    return len(set(s1)&set(s2));
```

- 나와 다른점 : set 자료형의 특징(중복x)을 살려서 도출한 결과. 쏘 스마트..
- 과정 :
  1. `set(s1)`, `set(s2)` : 리스트 s1을 **set**으로 변환
   
     → 중복 요소가 제거됨

  2. `set(s1) **&** set(s2)` : 두 집합의 교집합
   
        → 두 리스트에 공통으로 들어있는 원소들만 남음

  3. `len(...)` : 교집합의 원소 개수 세기
